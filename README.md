# 🍕 VITastic – Campus Discovery Platform

A fun and responsive web platform helping VIT Vellore students discover the best food spots and entertainment around Katpadi. Browse 27+ locations, get directions, view menus, and chat with an AI assistant for personalized recommendations.

> 🌐 **Live Site:** [VITastic on GitHub Pages](https://sithi-vignesh.github.io/VITastic/)

---

## ✨ Features

- **27+ Locations** — food spots and entertainment venues around VIT Vellore and Katpadi
- **Detailed Listings** — ratings, opening hours, popular dishes, contact info, menu links, and embedded Google Maps
- **Smart Filtering** — filter by cuisine type, tags, and location (Inside VIT / Around VIT / Inside Vellore)
- **AI Chatbot** — ask questions and get personalized location recommendations
- **Animated Landing Page** — falling food emojis on the homepage 🍕🍔🍣
- **Fully Responsive** — works on mobile and desktop
- **No Backend** — entirely frontend, deployed on GitHub Pages

```
│   ├── Home.html       # Main browse page
│   ├── Home.css/js     # Home page styles and logic
│   ├── Foodie.html     # Food listings page
│   ├── Foodie.css/js   # Food page styles and logic
│   ├── Gamez.html      # Entertainment listings page
│   ├── Gamez.css/js    # Entertainment page styles and logic
```

## 🚀 Getting Started

